Ce este Network-Calculator?

Network-Calculator este o pagina web ce permite procesarea diverselor calcule legate de relele. Aplicatia poate fi utilia studetiilor doritori sa invete mai multe despre retelistica, administratoriilor de retea pentru calculul rapid al subretelelor, sau oricine este interesat de atfel de calcule legate de adrese IP.

Pagina web a fost creata folosind HTML5 in combinatie cu CSS si Javascript. Codul sursa al acestei aplicatii si o prezentare PowerPoint poat fi gasite pe github la adresa de mai jos, alaturi de prezentarea video a aplicatiei.
